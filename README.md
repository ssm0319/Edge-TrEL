# Edge-TrEL Code 
Code for "Bridging Steady-State and Transient Carrier Dynamics in Light-Emitting Diodes via Edge-modulated Transient Electroluminescence"

This repository contains the code for Edge-TrEL methodology. 

# System Requirements 
* Python 
* Function generator (AFG 31022)
* Osciloscope (DPO 4054)
* PMT (Hamamatsu H10721-01)

# Instruction 
* Instrument.py: Equipment setting and connection method.
* Measurement.py: Data acquisition and slicing.
